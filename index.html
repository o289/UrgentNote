    テキストエリアの説明書

    テキストの追加

    テキストの追加するには文字を入力後、下にある「追加」を押すとテキストが追加されます。

    テキストの削除

    テキストの全削除

    テキストの全て削除するには文字を入力後、下にある「全削除」を押すと「テキストを全削除しますか」と聞かれるのではいを選択するとテキストが全て削除されます。

    テキストの個別削除

    各自使用しているデバイスのページをご覧くさい

    テキストBOXのリセット

    書いている文章をリセットする場合には下にある「リセット」を押すとリセットできます

    設定エリアの説明

    文字タイプ（フォント）

    こちらでは文字のタイプを選択することができます。

    文字サイズ

    こちらでは文字のサイズを選択することができます。

    文字の太さ

    こちらでは文字の太さを選択することができます。

    行間

    こちらでは行間の高さを選択することができます。

    リロードすると

    リロードすると設定エリアで設定した設定が全て初期設定に戻ります。

    初期設定

    初期設定は以下の通りです。

    フォント サイズ 太さ 行間
    1 中 中 低

    ショートカットキー一覧

    • command + A

      テキストの追加

    • command + D

      テキストの全削除

    • command + I

      テキストのリセット

    • command + F

      フォントをNoto Sans JPに変更

    • command + J

      フォントをNoto Serif JPに変更

    • command + W

      フォントサイズを24pxにする

    • command + E

      フォントサイズを36pxにする

    • command + R

      フォントサイズを48pxにする

    • command + Z

      フォントの太さをを細くする

    • command + X

      フォントの太さをを標準にする

    • command + C

      フォントの太さをを太くする

    • command + B

      行間を標準にする

    • command + N

      行間を少し高くする

    • command + M

      行間を高くする
